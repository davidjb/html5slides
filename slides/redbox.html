Tropical Data Hub (TDH) Features & Requirements

  • Researcher-friendly self-service interface: for less technical users
  • Institutional authentication: Mixed AAF and local authentication
  • Researcher profiles system: display public information, allows users to self-contribute details
  • Optimised for mobile and small-screen devices
  • ReDBox query interfaces available and documented
  • Other features as required by ANDS Metadata Stores project (MS07)

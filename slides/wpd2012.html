What is ?

  • At its most basic, it is a web-based CMS (Content Management System)
  • But a bit more than your normal CMS...
    • Built-in publication workflow
    • Designed for the web (document management vs content management)
    • Full-text document indexing
  • Dynamic, fast, scalable, secure, customisable
  • Open source (free), large community, long-lived (est. 2001)
  • Suitable for collaboration, Intranet, Internet sites and more

Add-on functionality (aka Pimp my Plone)

Flexible framework means anything can be modified in a reproducible manner. Some examples:

  • collective.geo: Geographic annotations for Plone (Example)
  • collective.easyslider: web-editable sliders (Example, Example 2)
  • PloneFormGen: User-editable web-based forms
    • (Example)

Add-ons open-source at github.com/collective

...and more (things that don't fit elsewhere)

  • Python (no PHP): better programming and excellent security: CMS Comparision
  • Runs anywhere (Windows, Mac, Linux); we host on Linux

  • Supports multiple authentication methods at once (local, LDAP/AD, OpenID, CAS...)

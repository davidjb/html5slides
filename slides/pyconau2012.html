Live-theming anything on the web using WSGI and Diazo (diazo.org)

  • Diazo's WSGI middleware uses rules (CSS3/XPath) to transform a response ("content") into a theme.
    • (XSLT minus pain)
    • (Can use live web pages as theme [is cached])
  • Run Diazo in front of a (gently modified) Paste proxy which talks to any app, HTML, website etc.
  • Front-end web server tells Diazo which theme to use (and where to proxy to).
  • Web server ↔ uWSGI (Diazo/Proxy) ↔ Anything
  • Configure everything from web server → Win!
  • Free bonus: change theme on any condition your web server supports
